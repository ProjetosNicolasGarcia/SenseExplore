Sobre o projeto

O SenseExplore oferece uma experiência imersiva por meio de um jogo de múltipla escolha em uma aplicação web totalmente planejada para o uso de deficientes visuais. Este jogo simula os desafios que os deficientes visuais enfrentam na web e no mundo, buscando promover empatia e incentivar práticas inclusivas na internet, proporcionando uma compreensão mais profunda sobre as barreiras enfrentadas e soluções para uma experiência web mais acessível a todos.

O que são deficientes visuais?

A deficiência visual é caracterizada pela perda ou redução permanente da capacidade visual em ambos os olhos, sem possibilidade de correção por meio de lentes, tratamentos clínicos ou cirúrgicos. Ela é dívidida em dois grupos principais: Baixa visão, onde a perda é leve ou moderada, provocando diminuição das respostas visuais mesmo com tratamentos. E cegueira, onde há ausência total de resposta visual, se fazendo necessário o uso do sistema Braile para ler e de bengalas para se locomover.

Como eles utilizam a internet?

Para acessar a web através de dispositivos desktop, os deficientes visuais, utilizam técnologias assistivas que tornam a navegação online possível, umas das principais ferramentas para isso, são os Softwares de leitura de tela, que convertem o texto exibido em áudio, alguns exemplos são o NVDA, software que lê textos através de comandos no teclado e o DosVox, sistema operacional da UFRJ que possui um navegador próprio adaptado para receber comandos de teclado. Além disso, existem outras alternativas, como teclados adaptados com braile e tecnologias de comando por voz, para a utilização em dispositivos mobile.
